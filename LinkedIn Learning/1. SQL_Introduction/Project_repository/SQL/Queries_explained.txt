Queries Explained

1. Basic SELECT Statement
SELECT 'Hello, World!' AS greeting;

This query demonstrates the most basic form of a SQL SELECT statement.

The query returns a single text value and assigns it the column name greeting using an alias.

Purpose:

Demonstrates SQL query execution

Introduces column aliasing using AS

Confirms that the SQL client is working correctly

Expected result:

greeting
Hello, World!
2. Arithmetic Expression Example
SELECT 2 + 2 AS addition_result;

This query demonstrates that SQL can perform mathematical calculations directly within a SELECT statement.

Purpose:

Shows that SQL can evaluate expressions

Demonstrates simple arithmetic operations

Files and Descriptions


Introduces result aliasing

Expected result:

addition_result

3. PostgreSQL Version Check
SELECT version();

This query retrieves the current PostgreSQL server version and system build information.

Purpose:

Confirms connection to a PostgreSQL database

Displays the installed PostgreSQL version

Useful for debugging and environment verification

Example output:

PostgreSQL 18.x on x86_64-pc-linux-gnu, compiled by gcc, 64-bit
